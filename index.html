À propos

Bienvenue sur mon tout premier serveur Minecraft Mini-Jeux ! Développé sur la version 1.21.1 Java et hébergé via Aternos, il est fluide, fun, et accessible à tous grâce à Geyser qui permet de jouer en Java et Bedrock ensemble !

Mini-Jeux

🕵️‍♂️ Cache-cache : Une map remplie de cachettes pour bien galérer à trouver les autres !
💣 Déaoudre : Jeu original avec 3 niveaux croissants en difficulté.
🏎️ Carting : Courses folles entre amis pour tester vos réflexes !
⚔️ PvP : Une arène compacte pour des duels ou des 3v3 épiques.
💥 Creeper Hunt : Une plateforme difficile remplie de creepers à éviter ou combattre !

Plugins utilisés

  • Chunky
  • CommandPanels
  • EventSounds
  • Floodgate
  • Geyser-Spigot
  • GeyserSkinManager
  • LuckPerms
  • SkinsRestorer
  • bStats / PluginMetrics
  • AUpdater

Compatibilité

🖥️ Le serveur est compatible Java & Bedrock grâce à Geyser. Peu importe votre version, vous pouvez jouer ensemble !

🌐 Héberger la map gratuitement sur Aternos

Voici comment héberger facilement le serveur chez toi avec Aternos, une plateforme gratuite et fiable :

  1. Va sur : https://aternos.org/ et connecte-toi ou crée un compte.
  2. Crée un serveur Java en choisissant la version Spigot/Bukkit 1.21.1.
  3. Va dans “Monde” → clique sur “Téléverser” et choisis le fichier ZIP de la map :
    • 🔗 Télécharger la map
  4. Rends-toi dans “Plugins” et ajoute ceux-ci :
    • ✔ Geyser-Spigot + Floodgate
    • ✔ LuckPerms
    • ✔ SkinsRestorer
    • ✔ Chunky, CommandPanels, EventSounds...
  5. Redémarre ton serveur 🟢

✨ Tu es maintenant prêt à jouer sur ton propre serveur mini-jeux, en solo ou avec tes amis !

Java et Bedrock sont compatibles automatiquement grâce à Geyser.
